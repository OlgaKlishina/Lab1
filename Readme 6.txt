1. Для фунционирования программы необходимо установить пакет eslint, прописав в командной строке: npm install eslint —save-dev.
2. Чтоб запустить программу, прописываем в командной строке node lab6.js и заходим на адрес localhost:3000/register4.html.
3. Пишем любой код в столбец, например:
var a
var b
4. Результатом работы программы станет добавление точки с запятой ";" после инициализации переменной.
