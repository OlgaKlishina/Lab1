Для решения текущей задачи потребовалось совершить следующие шаги:
1. Написать программу на node.js.
2. Создать форму html.
3. Осуществить взаимодействие между формой и программой.
4. Устанавливаем компонент Express Formidable для осущесвления передачи данных между формой и программой. 
Запуск программы осуществляется следующим образом:
1. Программу и форму помещаем в папку, где хранится модуль Express Formidable.
2. Прописываем в командной строке путь к данной папке.
3. Запускаем программу командой node form.js.
4. Заходим на http://localhost:3000/. 
5. В появившейся форме вводим имя пользователя.
6. Нажимаем на кнопку "Создать профиль".
7. В качестве результата выполнения программы получаем приветсвие, адресованное данному пользователю.
